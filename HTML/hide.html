Hide (documentation)

Hide is a feature built in to Virtual to quickly toggle from games to school work.

The main purpose of hide is to avoid being caught on Virtual. On the right, there are a few buttons. First, we have the HIDE button. This button immediately hides your screen using the settings you have underneath. Then, we have the toggle hide key. This allows you to pick any key on your keyboard so that when it is pressed, the hide function executes automatically. Finally, there are the different hide modes. You can choose between immediately closing the page, opening a new one, both, or an image overlay.

Once you set a hide key, it becomes effective immediately. If that key is pressed again without any new modifications, it will hide. Virtual recommends you choose the close page option for hiding as it is the most effective & doesn't require time to load any additional new pages. Lastly, the HIDE button in the hide page is only meant for testing purposes. Virtual recommends you set a toggle hide key for the most effectiveness.

Never be too safe with the hide feature. Anyone - teachers, classmates, & even your own friends can be what gets you sold out. Use the hide feature wisely & as recommended for most effectiveness. Note that the open new tab feature will not work unless you paste in a full, real URL. We ask you to paste the URL in (by right clicking & selecting paste) so you don't accidentally activate the hide feature while typing in the URL.
